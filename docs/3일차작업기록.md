## Docker 확인 결과

`docker compose config`로 Compose 문법은 검증했습니다.

Docker Desktop 실행 후 `docker compose up -d`로 MySQL/Redis 컨테이너 실행을 확인했습니다.

```text
seathub-mysql   mysql:8.4   Up (healthy)   3306
seathub-redis   redis:7.2   Up (healthy)   6379
```

Redis는 `redis-cli ping`으로 `PONG` 응답을 확인했고, MySQL은 `mysqladmin ping`으로 `mysqld is alive`를 확인했습니다.
